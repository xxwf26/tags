# style-atlas · 画风作品库

一个以「画风」为索引的作品发现引擎。按多维画风标签检索作品并反查画师；库里不够时，按画风需求去小红书 / 微博 / 米画师主动爬取作品和画师，AI 自动打标，人工复核后入库或建联。

- 完整产品设计方案：[docs/画风作品库设计方案.md](docs/画风作品库设计方案.md)

## 定位

从「管画师」升级为「以画风找作品 → 反查画师」。核心对象从画师下沉到**作品**：一个画师 N 个作品，画风标签打在**每个作品**上（多维、细粒度），支持交叉筛选。

## 技术栈（规划）

- 前端：React 19 + TS + Vite + Tailwind4 + shadcn/ui + react-query
- 后端：NestJS 10 + Drizzle ORM
- 数据库：MySQL 8（库名 `style_atlas`）
- 端口：后端 `3200`，前端 dev `5300`

## 分期路线

| 阶段 | 交付 |
|---|---|
| P0 地基 | 脚手架 + 核心表 + 标签维度种子 + 手动录作品 + 画风画廊多维筛选 |
| P1 AI 打标 | 视觉打标管线 + 白名单归一 + 复核态 + 存量批量打标 |
| P2 外部采集 | crawl 管线（小红书先行）+ 候选复核队列 + 转正入库 + 画画习惯聚合 |
| P3 增强 | 以图搜图 + 微博接入 + 周期重爬 + 去重 |
